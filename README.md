# clase2.0
Markdown es básicamente un lenguaje de marcado ligero que sirve para darle formato a un texto usando caracteres especiales de forma súper sencilla.

### 1. Títulos y Estructura

Se usan almohadillas o "hashtags" (#). Mientras más pongas, más chico es el título.

* `# Título Grande`
* `## Subtítulo`
* `### Título de sección`

### 2. Énfasis (Formato de texto)

* **Negrita**: Envolvé el texto con dos asteriscos: `**texto**`.
* *Cursiva*: Usá un solo asterisco o guion bajo: `*texto*` o `_texto_`.
* ~~Tachado~~: Usá dos virgulillas: `~~texto~~`.

### 3. Listas

* **Desordenadas**: Usás un asterisco o un guion.
* `* Item A`
* `* Item B`


* **Ordenadas**: Simplemente ponés el número seguido de un punto.
1. Primero
2. Segundo



### 4. Links e Imágenes

* **Links**: `[Nombre del link](https://url.com)`
* **Imágenes**: Es igual, pero con un signo de exclamación al principio: `![Texto alternativo](url_de_la_imagen)`

### 5. Código y Bloques

* Para resaltar una palabra como `código`, usás comillas simples invertidas: ``código``.
* Para bloques enteros de código, usás tres comillas invertidas (```) arriba y abajo.

En **Markdown** puedes crear tablas usando **barras verticales `|`** para separar columnas y **guiones `-`** para definir el encabezado.

### Ejemplo básico

```markdown
| Nombre | Edad | Ciudad |
|-------|------|--------|
| Ana   | 25   | Madrid |
| Luis  | 30   | México |
| Sofía | 28   | Buenos Aires |
```

**Resultado:**

| Nombre | Edad | Ciudad       |
| ------ | ---- | ------------ |
| Ana    | 25   | Madrid       |
| Luis   | 30   | México       |
| Sofía  | 28   | Buenos Aires |

---

### Alineación de columnas

Puedes alinear el texto usando `:` en la fila de separación.

```markdown
| Producto | Precio | Stock |
|:---------|:------:|------:|
| Laptop   | 1200   | 5     |
| Mouse    | 25     | 40    |
| Teclado  | 80     | 12    |
```

* `:---` → izquierda
* `:---:` → centro
* `---:` → derecha

**Resultado:**

| Producto | Precio | Stock |
| :------- | :----: | ----: |
| Laptop   |  1200  |     5 |
| Mouse    |   25   |    40 |
| Teclado  |   80   |    12 |
